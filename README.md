## 目标与哲学

众所周知，前端发展如火如荼，日新月异，而且很长一段时间内都将如此。这份由[晚晴幽草轩轩主](https://jeffjade.com)正在维护的教程列表；旨在为`前端学习`，`技能提升`，`视野扩展`，`资料查找`等行个方便。将会在日常工作探索、学习中，持续保持更新，增新优之章，除旧失之文；尽可能使得这份列表保持简练，以避免信息过重且冗杂；望其能在浩如烟海的讯息中，堪有所用。
